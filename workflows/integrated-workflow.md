# Integrated Workflow (v5)

## Full Lifecycle with Feedback Loops

### Phase 1: CBO Discovery
```
CEO → Pain Point Hunter → Competitor Analyst → BM Designer → Business Planner → GTM Strategist
                                                                                     ↓
                                                                            Product Brief
                                                                                     ↓
                                                                    ── Gate 1 체크 ──
                                                                   ↓ PASS        ↓ FAIL
                                                              Phase 2         CBO 보완 (max 2회)
```
**소요 시간**: Tier 2: 4h | Tier 3: 16h

### Phase 2: CTO Development
```
PRD Architect → Scope Guard → Architect → Frontend + Backend (병렬) → Code Reviewer → DevOps
                    ↑                                                        ↓
                    └── 범위 초과 시 PRD 수정 ─────────────────────────────────┘
                    
 * 기술 불가 발견 시: CTO → CBO 롤백 (max 2회)
```
**소요 시간**: Tier 2: 40h | Tier 3: 160h

### Phase 2.5: CDO Design
```
CTO → Design Upgrade Brief (Gate 3) → CDO
                                        ↓
UI Designer → Visual Director → Image Generator → Screenshot Designer → ASO Optimizer → D2C Bridge
        ↕ (CTO↔CDO iteration max 3회)                                          ↓
                                                                        Design Delivery (Gate 4)
                                                                        Store Assets Package
```
**소요 시간**: Tier 2: 8h | Tier 3: 40h

### Phase 2.9: User Validation Gate
```
5명 대상 Usability Test
    ↓
Task Completion Rate ≥ 70%? ──→ NO: CDO+CTO 수정 (max 2회)
    ↓ YES
Core Value 이해율 ≥ 80%? ──→ NO: Onboarding 재설계
    ↓ YES
NPS ≥ 7? ──→ NO: 핵심 불만 수정
    ↓ YES
Phase 3 진행
```
**소요 시간**: 16h (Tier 3에서만, Tier 2는 스킵 가능)

### Phase 3: CGO Growth
```
Acquisition Strategist → Ads Operator + Content Creator (병렬)
                              ↓
Onboarding Designer → Retention Manager → Revenue Optimizer
                              ↓
Copy Strategist (전 과정 지원) + Growth Analyst (데이터 수집)
                              ↓
                    Growth Insights (Gate 6) → CEO Go/No-Go
```
**소요 시간**: Tier 2: 16h | Tier 3: 80h

### Phase 4: CEO Decision
```
Growth Analyst의 Go/No-Go Assessment 기반:
    CONTINUE: 현재 전략 유지 + 스케일
    ITERATE: CGO + CTO 개선 사이클 (2주 단위)
    PIVOT: CBO 재검토 (같은 타겟 다른 솔루션 or 같은 문제 다른 접근)
    KILL: 프로젝트 종료 + 학습 기록
```
